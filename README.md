# Full Stack Task Tracker

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Docker

## Features
- User Registration
- Login with JWT
- Protected Routes
- Create / Delete Tasks
- User-specific Tasks

## How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start

### MongoDB
docker run -d -p 27017:27017 --name taskmongo mongo
